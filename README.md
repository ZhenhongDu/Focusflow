# 安装使用

## 本地使用

1. 从github复制仓库
```
git clone https://github.com/ZhenhongDu/Focusflow.git
```

2. 安装 node.js
3. 安装依赖包
`npm install`
4. 本地启动
`npm run dev`
5. 浏览器打开
```
http://localhost:3000/
```
