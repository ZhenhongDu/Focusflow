## ✨ 功能特性
Focusflow 是一款基于 Web 的任务管理与专注时间追踪工具，致力于成为 Web 版的 "3x3" 体验。

### 🚀 任务管理 (Todo)

* **🗂️ 任务分组:** 清晰归类你的所有待办事项。

* **⏳ 专注计时:** 针对特定任务开启专注计时器。

* **🎬 场景功能:** 快速启动常用的专注事项。

* **✍️ 补记时长:** 轻松补全遗漏的专注时间。

* **🕒 时间轴视图:** 直观查看你的专注时间流。

* **🏷️ 标签管理:** 灵活地为你的任务和时间打上标签。

* **👀 任务时长追踪:** 跟踪每个 Todo 任务上花费的专注时间。

* **📈 标签时间分布:** 分析不同标签下的时间投入情况。

### 📊 专注分析 (Analysis)

* **💡 专注时间分析:** 深入了解你的时间使用模式。

* **🔖 标签数据洞察:** 查看各个标签的时间占比与详情。

* **🔢 专注详情统计:** 获取详细的专注数据报告。

* **📚 详细专注记录:** 永久保存你的每一次专注。

* **📉 专注趋势图:** 观察你的专注习惯随时间的变化。

* **🗓️ 每周热力图:** 以周为单位，查看你的高效时间分布。

* **🔥 年度贡献热力图:** 概览全年的专注时长与投入。

## 🛠️ 安装与使用

### 本地启动

请按照以下步骤在你的本地环境中运行 Focusflow：

1. **克隆仓库**

   ```bash
   git clone https://github.com/ZhenhongDu/Focusflow.git
   ```
2. **安装 Node.js**


确保你的电脑已经安装了 Node.js (LTS 版本)。

3. 安装依赖 
进入项目目录并执行：
```Bash
npm install
```
4. 本地启动 
运行开发服务器：
```Bash
npm run dev
```
5. 访问应用 
在你的浏览器中打开：

http://localhost:3000/
